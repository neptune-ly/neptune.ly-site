What it isما هو

Multi-bank payment gatewayبوابة دفع متعددة المصارف

Neptune Astro is our payment gateway built on the OpenWave specification. It routes payments across member banks and wallets, manages aliases, merchants, settlements and webhooks, and gives every participant the same clean API surface. astro-sdk — public on GitHub — lets fintechs and merchants integrate in days instead of months.

Neptune Astro هي بوابة الدفع التي بنيناها على مواصفة OpenWave. توجّه المدفوعات بين المصارف والمحافظ الأعضاء، وتدير الأسماء المستعارة والتجّار والتسويات وWebhooks، وتمنح كل مشارك سطح API واحداً نظيفاً. أما astro-sdk — المتاحة للعموم على GitHub — فتمكّن شركات التقنية المالية والتجّار من التكامل خلال أيام بدل شهور.
